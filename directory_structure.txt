# Databases


1. SNP Information
A table to store the SNP information include SNP names, genomic coordinates (chromosome, start, end), and possibly mapped gene names.

Table: snp_db

snp_id (Primary Key)
rs_value (SNP identifier)
chromosome
start_position (Genomic start position)
end_position (Genomic end position)
mapped_gene (Gene associated with the SNP, if any)
p_value (P-value from the association test)

2. Positive Selection Summary Statistics
A table to store summary statistics for positive selection for each SNP, for different populations. This could be a table with multiple entries per SNP (i.e., multiple summary statistics for the same SNP).

Table: posel_db

stat_id (Primary Key)
snp_id (Foreign Key to snp_info table)
population 
stat_type (e.g., Fst, Tajima's D, etc.)
statistic_value
statistic_description

3. Gene Information
Functional or ontology terms to describe genes. Related to the mapped_gene column in the snp_info table.

Table: gene_db

gene_id (Primary Key)
gene_name 
ontology_term

4. Population Information
Descriptive information about populations (such as geographical sampling locations)

Table: population_db

population_id (Primary Key)
population_name
sampling_location (Geographical location where the population was sampled)
population_size 
study_reference ?

5. Plot Data
Dynamic plotting and visualising summary statistics across genomic regions.

Table: plot_db

plot_id (Primary Key)
snp_id (Foreign Key to snp_info table)
population_id (Foreign Key to population_info table)
region_start (Start of genomic region)
region_end (End of genomic region)
summary_statistic_value (The summary statistic value to be plotted)
plot_type (Type of plot, e.g., "heatmap", "scatter", etc.)
